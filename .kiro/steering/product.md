# Product Overview

AccessMotor is a Spring Boot e-commerce web application for selling flowers and floral arrangements. This is a school project that demonstrates a complete web application with user authentication, product catalog management, shopping cart functionality, and order processing.

## Key Features

- **Product Catalog**: Browse flowers by categories (wedding flowers, bouquets, gift baskets, etc.)
- **User Management**: Registration, login, and user profiles with role-based access
- **Shopping Cart**: Add products to cart and checkout functionality
- **Admin Dashboard**: Manage products, categories, orders, discounts, and user accounts
- **Order Management**: Track and process customer orders
- **Security**: Spring Security integration with authentication and authorization

## Target Users

- **Customers**: Browse and purchase floral products
- **Administrators**: Manage the store inventory, orders, and customer accounts

## Business Domain

The application focuses on the floral industry with Vietnamese language support, handling various flower categories and arrangements for different occasions (weddings, celebrations, condolences, etc.).
